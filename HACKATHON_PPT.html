The Innovation

TensorX replaces sequential form-processing with Parallel Agentic Execution. 75 agents vet the applicant in real-time, delivering a loan decision faster than a search query.

03. The Problem Landscape

Why traditional digital lending is broken.

Systemic Friction

Traditional onboarding is fragmented. Data resides in silos, verification is manual, and the user experience is designed for the lender, not the applicant. TensorX reverses this paradigm.

⚠ 67% Abandonment Rate
⚠ 3-5 Days Disbursement Delay

04. Solution: Aria AI Agent

The first truly autonomous voice call onboarding experience.

Conversational Verification

Aria replaces friction with flow. Applicants simply talk to the AI agent via WebRTC.

  • Multilingual: Converses in Hindi, English, and 5 regional languages.
  • Liveness-First: Continuous video anti-spoofing during the call.
  • Field Extraction: Auto-fills PAN, Aadhaar, and Income details.

05. Technical Architecture

Built for enterprise-grade scalability and sub-100ms latency.

The Tech Stack

Backend: FastAPI, Python
Streaming: LiveKit WebRTC
Frontend: Next.js, Tailwind
Database: PostgreSQL, Redis
Models: YOLO, Whisper, LLM
Deployment: Docker, Kubernetes

Agentic Workflow

Our orchestrator manages 75 parallel micro-agents via a stateless WebSocket pipeline, ensuring zero data loss and instant state synchronization across the applicant and admin interfaces.
